# BookEase

Brief description: BookEase is a responsive web application that simplifies appointment booking for service providers and their clients. Users can easily select a service, choose a preferred date and time, and enter their contact details. The app ensures valid booking times by disabling past hours and provides a convenient way to confirm appointments via WhatsApp.

Built with HTML, CSS, and Vanilla JavaScript, BookEase is fully responsive, user-friendly, and ready for deployment on platforms like Vercel. It demonstrates practical skills in UI/UX design, form validation, and dynamic DOM manipulation.

## Features
Select from multiple services with duration and price information.

Date picker with minimum date restriction.

Time picker that disables past hours dynamically.

Real-time booking summary.

Confirmation via WhatsApp.

Responsive design for desktop and mobile devices.
